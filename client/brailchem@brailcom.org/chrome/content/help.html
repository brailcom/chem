What is Brailchem

Brailchem is a studying tool for blind and visually impaired students of chemistry. It is developed as a part of the ICT project.

Usage

Brailchem consists of several subapplications described below. All basic commands are available in the menus.

Global command keys:

Control+QQuit Brailchem window

Periodic Table Application

Periodic table provides information about elements and their properties. You can navigate through it using arrow keys together with Shift modifier. Pressing an element button with the Enter key shows complete information about the given element.

Periodic table command keys:

Shift+arrow keysnavigate through elements of the periodic table
Control+Tgo to the main periodic table
Control+Shift+Tgo to the table of lanthanoids and actinoids
Control+Ejump to element
Control+F Ego to electronegativity filtering
Control+F Ggo to group filtering
Control+F Ogo to oxidation number filtering
Control+F Sgo to set of elements filtering
Control+F ?show currently filtered elements

You can limit browsable elements by setting filters. When filters are enabled, you can browse only elements satisfying all given filtering conditions.

A special filter is the Set of elements filter. You activate it by pressing the Change button in the Set of elements filter. Then you are directed to the periodic table when you can insert or remove elements into or from the set by pressing their buttons. The following additional command keys are available during element selection:

Control+S dotfinish element selection and activate it
Control+S Aselect all currently active elements
Control+S Ndeselect all elements

Molecule Browser Application

Right now this application serves mostly as a proof of concept of communication between the client and the server. This application is in development and will change significantly.
