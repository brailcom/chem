Basic chemistry terms for users of Brailchem software

Periodic table

Symbol

Symbol of element is derived from its latin name. It is used in written and oral communication, symbols are used to create molecular formulas and describe reactions. Symbol is the most important part of information about an element.

Atomic (proton) number

Number of protons in atom nucleus. It is often designated by the symbol Z. Atomic number unambiguously determines the element and number of its electrons in uncharged state.

Column

Groups of elements designated by roman numbers I. – VIII. and letters A and B. There are 16 columns in the periodic table, divided to non-transition elements (I. – VIII. A) a transition  elements (I. – VIII. B). Elements of the same column have the same number of valence electrons (electrons available for reaction) and therefore similar chemical properties.

Row

Rows of periodic table are more often referred to as periods. There are seven periods. Periods are ordered by increasing value of the proton number and therefore atomic weight – lightest elements in the first row, heaviest in the last.

Element group

Assignment of elements into groups of metals, nonmetals, halogens, noble gases, alkali metals, alkali earth metals. This partitioning partially corresponds with columns of periodic table and again groups elements of similar properties.

Oxidation number

Sum of positive and negative charges in an atom. Atoms in ground state have oxidation number of 0. Each element has a typical set of oxidation numbers; their knowledge help in determining reaction outcome.

Electron configuration

Describes organization of electrons in electron shell. It is important in assessment of chemical properties of an element.

Atomic weight

Relative weight of one atom of given element against standard (atomic weight constant –  1/12 of weight of one carbon atom, mu = 1,66057*10-27 kg). It allows description of atomic weights in "reasonable" units from 1 to about 300. With rising proton number the atomic weight generally increases.

Electronegativity

Ability of an atom to attract bonding electrons. This value is typical for each element and enables chemists to assess its behavior in specific types of reactions and also properties of its compounds.

Description

General information about an element, e.g. its color, reactivity, assignment between metals, nonmetals, semi-metals and gases.

 

Structure viewer

ChemDraw binary format

Binary format used by the popular ChemDraw program and supported by several other programs for chemical drawing.

ChemDraw CDXML formát

XML version of the above described binary format.

Chemical Markup Language

Standardized XML format for description of molecules which is also often used in Internet sources.

Linear format suitable for unambiguous "naming" of molecules in a computer readable form. Lately more and more often used in Internet sources.

Molfile

Probably the most widely used standardized format for recording of molecular structure. Its is used by many databases as conversion format and is often available in Internet sources.

Linear format similar to classical notation for description of chemical molecules. It was devised to be both human readable and writable, e.g. CC=O (ethanal). This format makes it possible to create a description of a molecular structure without a need for a drawing program and is therefore especially suitable for use in written text and on the Internet.

Summary formula

Describes types and numbers of atoms in a molecule of given compound. It is written down as a series of element symbols and their count. It does not allow description of the internal structure of a molecule, just its composition.

Protein Data Bank format

Text format for description of biological macromolecules (proteins, nucleic acids).

Charge

Difference between the number of electrons and protons in a molecule – when there are more electrons than protons, the charge is negative, otherwise it is positive.

Molecular weight

Sum of atomic weights of all atoms in a molecule.

Single bond

Bond formed by one pair of electrons, written as a single line (-). Bond order is a very important factor in determining bonds reactivity and other properties.

Double bond

Formed by two electron pairs, written as two parallel lines (=).

Triple bond

Formed by three electron pairs, written as three parallel lines (≡).

Functional group

Atom or a group of atoms of specific structure that brings characteristic properties to the molecules, such as reactivity in some kinds of reactions. Determination of functional groups a inevitable for understanding of chemical properties of a molecule.
