O que é o Brailchem

Brailchem é uma ferramenta para ajudar os estudantes cegos e com baixa visão a estudarem química. Os estudantes cegos têm de conviver com dificuldades especiais durante o seu estudo tal como a compreensão de informação normalmente apresentada de forma visual (por exemplo as estruturas de moléculas orgânicas) ou aprendizagem de relações estruturais complicadas sem o apoio da sua representação visual (por exemplo a tabela periódica dos elementos). O objectivo do Brailchem é a disponibilização de informação química usando computadores de uma forma acessível aos estudantes cegos e com baixa visão. 

O Brailchem foi desenvolvido no âmbito do  projecto ICT.

Utilização


O Brailchem é constituído por duas sub-aplicações: o editor molecular e a tabela periódica dos elementos químicos. As sub-aplicações podem ser iniciadas através dos itens correspondentes no menu da Aplicação. O utilizador pode correr cada uma das aplicações numa ou mais janelas separadas, criadas a partir do menu Ficheiro / Nova Janela.

O Brailchem é uma aplicação orientada para a Internet. Consiste em duas partes: um cliente e um servidor. O servidor, basicamente invisível ao utilizador, fornece todos os dados químicos ao cliente. O cliente não tem informação própria sobre os elementos químicos ou a estrutura das moléculas, é apenas uma ferramenta de apresentação dos dados fornecidos pelo servidor. Desta forma, para funcionar, o cliente deve estar ligado a Internet e ter acesso ao servidor do Brailchem. Este servidor está definido à partida na aplicação e é usado por defeito; se preferir utilizar outro servidor, é possível defini-lo no diálogo das preferências do cliente. Por favor, utilize apenas servidores de confiança, de outra forma arrisca-se a ter problemas de privacidade.

Para começar a usar o Brailchem, navegue nos seus menus, leia a ajuda disponível e tente usar as sub-aplicações. Todas as informações importantes sobre o estado actual do programa e as acções executadas são registadas na barra de estado da janela da aplicação.

Por agora existe apenas uma única tecla especial de comando global no Brailchem:
Control+QFechar a janela do Brailchem

Naturalmente, pode também usar os comandos do menu de acesso. Outros comandos são específicos de cada uma das sub-aplicações do Brailchem. 

Nas seguintes secções descrevem-se as sub-aplicações do Brailchem.

Aplicação Editor Molecular

O editor molecular é capaz de analisar as estruturas moleculares e as reacções químicas escritas em qualquer um dos sistemas de notação química suportados pelo servidor ou identificados pelo seus nomes. A molécula ou a reacção é escrita na caixa de entrada da aplicação. Alternativamente pode escrever o URL da localização que contem a molécula ou a descrição da reacção na caixa de entrada. file e http Os URLs são suportados.

De seguida especifique o formato de entrada na caixa combinada seguinte e pressione o botão de análise. Se a molécula de entrada é reconhecida, é originada uma descrição editável e expandida dessa dada molécula, ou da reacção química que consiste de diversas moléculas, na janela da aplicação.

A aplicação funciona através da ligação ao servidor do Brailchem e solicitação da informação representada por uma dada sequência de SMILES. O servidor tenta identificar os nomes das moléculas e processa a expansão das moléculas como uma sequência de átomos e fragmentos ligados por ligações químicas. Os fragmentos são conjuntos de átomos definidos e identificados pelo servidor (por exemplo grupo carboxílico) que podem ser representados por um único item em vez de uma lista desnecessária e complexa de átomos individuais. 

Quando uma molécula é analisada (por exemplo, o ácido acético é escrito como CC(=O)O na notação SMILES), aparece: o seu nome, uma lista de propriedades básicas (tais como a fórmula molecular ou o peso molecular) e a sequência dos átomos e dos fragmentos. Para cada átomo ou fragmento são indicados todos os itens ligados à este através de ligações químicas. Estes itens podem ser de dois tipos: elementos anexos que se ligam apenas a este átomo e a nada mais (logo não há necessidade de saltar para eles para explorar a estrutura da molécula) e vizinhos que podem estar ligados a vários átomos.

Existem resultados especiais para alguns átomos e para as propriedades das ligações:

  • Se um dado átomo faz parte de um anel ou de um grupo de anéis, isto é indicado após o símbolo do átomo pelo R caracteres, um R caracter por cada anel.
  • Se a multiplicidade de um dado átomo é diferente de 1, isto é indicado após o símbolo do átomo pelo símbolo xNÚMERO onde NÚMERO é a multiplicidade do átomo.
  • As ligações aromáticas são indicadas pela letra A no fim da sua descrição.

Pode navegar através dos itens usando os comandos TAB e Shift+TAB, comandos de navegação do Firefox ou usando um dos comandos especiais desta aplicação:

Control+Asaltar para os átomos da molécula; usando esta chave pode saltar do nome da molécula directamente para a lista de átomos e de fragmentos sem passar através das propriedades da molécula
Control+Nir para o átomo seguinte
Control+Pir para o átomo anterior
Alt+Msaltar para caixa de entrada da molécula
Alt+Galternar com a opção apresentação de grupos de átomos (se aplicável)

Adicionalmente quando pressiona Enter em qualquer dos nomes dos elementos anexos ou dos vizinhos, o enfoque passa para o elemento referido.

Se a molécula indicada contem fragmentos, é adicionada uma caixa de opção no final. Esta caixa de opção permite definir a forma como são apresentados os grupos de átomos da molécula. Existem duas opções: indicação dos fragmentos, em vez da sua representação em determinados grupos de átomos, ou a simples indicação dos átomos, sem referência aos fragmentos.

Ao analisar uma reacção química global em vez de uma única molécula (por exemplo CC(=O)O.CCO>[H+]>CC(=O)OCC.O na notação SMILES), a sua apresentação é um pouco mais complexa. Inicialmente é apresentado um resumo dos reagentes, dos reagentes químicos e dos produtos. Pressionando na tecla Enter em qualquer dos nomes dos diferentes componentes da reacção, a análise salta para a descrição da molécula correspondente. As moléculas são então apresentadas de uma maneira semelhante  ao que acontece para uma única molécula.

Podem ser utilizadas as mesmas teclas de comando usadas para a análise de uma molécula. As teclas que permitem navegar nos átomos encontram-se contudo limitadas à molécula em análise. São definidas  teclas adicionais de comando para permitir navegar entre moléculas:

Control+Arrow Downir para a molécula seguinte 
Control+Arrow Upir para a molécula anterior
Control+Tsaltar novamente para o resumo da reacção

Aplicação Tabela Periódica 

A tabela periódica fornece informação sobre os elementos químicos, as suas propriedades e relações. Pode navegar através dela usando as teclas de seta enquanto pressiona a tecla Shift. Pode também aceder a toda a informação sobre um dado elemento pressionando o botão do elemento com o comando Enter.

Com o objectivo de facilitar a orientação procedeu-se à divisão tabela periódica em duas partes, como é habitual: A tabela periódica dos elementos principais e a tabela das séries dos lantanídeos e dos actinídeos. Pode saltar para a tabela dos elementos principais pressionando o comando Control+T e para a tabela das séries dos lantanídeos e dos actinídeos pressionando o comando Control+Shift+T.

Sumário das teclas comando da tabela periódica:

Shift+arrow keysnavegar através dos elementos da tabela periódica
Control+Tsaltar para a tabela periódica dos elementos principais
Control+Shift+Tsaltar para a tabela periódica das séries dos lantanídeos e dos actinídeos
Control+Esaltar para um dado elemento

Pode limitar a apresentação da tabela periódica usando filtros. Os filtros podem reduzir o conjunto de elementos da tabela periódica apresentados, apenas aqueles que satisfazem critérios baseados em propriedades dos elementos (tais como o grupo ou o número da oxidação). Os filtros podem ser definidos na caixa de Filtros, situada abaixo da tabela periódica. Se definir mais de um filtro, apenas são apresentados os elementos que satisfazem simultaneamente todas as condições impostas.

Pode usar os seguintes atalhos de teclado para trabalhar com filtros:

Control+F dotir para os filtros
Control+F Eir para a filtração por electronegatividade
Control+F Gir para a filtração por grupo
Control+F Oir para a filtração por número de oxidação
Control+F Sir para a filtração por conjunto de elementos
Control+F ?mostrar os elementos filtrados

Estes atalhos são constituídos por uma sequências de algumas teclas. Pressione em primeiro lugar o comando de Control+F e depois a segunda tecla.

Um filtro especial é o filtro Conjunto dos elementos filtrados. Pode activa-lo pressionando o botão de Mudança no Conjunto dos elementos filtrados. É então direccionado para a tabela periódica, onde pode introduzir ou remover elementos, de ou para o conjunto de elementos de trabalho pressionando os seus botões com a tecla Enter. As seguintes teclas de comando adicionais estão disponíveis durante a selecção do conjunto de elementos:

Control+S dotterminar a selecção do elemento e activá-la
Control+S Aseleccionar todos os elementos activos no momento
Control+S Nanular a selecção de todos os elementos
