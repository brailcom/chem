Termos químicos básicos para os utilizadores do software Brailchem

Tabela Periódica

Símbolo químico

O símbolo de um elemento químico deriva do seu nome em latim. É usado tanto na comunicação escrita como na oral, os símbolos químicos são usados para criar as fórmulas moleculares das moléculas e para descrever as reacções químicas. O símbolo químico é a parte mais importante da informação sobre um dado elemento químico.

Número atómico

O número atómico é o número de protões no núcleo de um átomo. É representado pelo símbolo Z. O número de protões determina sem ambiguidade o elemento químico e o número de electrões no seu estado fundamental (sem carga).

Coluna

Os grupos dos elementos químicos são designados pelos números romanos I - VIII e pelas letras A e B. Existem 16 colunas na tabela periódica, divididas em elementos químicos representativos (I – VIII A) e elementos químicos de transição (I – VIII B). Os elementos químicos na mesma coluna apresentam o mesmo número de electrões de valência (electrões disponíveis para reacção) e consequentemente propriedades químicas semelhantes.

Linha

As linhas da tabela periódica são habitualmente chamadas de períodos. Existem sete períodos. Os períodos são ordenados por ordem crescente de número atómico e consequentemente de peso atómico – os elementos químicos mais leves no primeiro período e os mais pesados no último período.

Grupo do elemento químico

Os elementos químicos são organizados por blocos, metais, não-metais, halogéneos , gases nobres, metais alcalinos e metais alcalino terrosos. Esta divisão agrupa colunas da tabela periódica e de novo grupos de elementos com propriedades semelhantes.

Números de oxidação

O número de oxidação  é a soma das cargas positivas e negativas de um átomo. Os átomos no estado fundamental tem o número de oxidação zero. Cada elemento químico tem um conjunto característico de números de oxidação; o conhecimento destes ajuda na previsão das reacções do elemento.

Configuração electrónica

Descreve a distribuição dos electrões em camadas electrónicas ou níveis de energia. É importante na avaliação das propriedades químicas de um elemento químico.

Peso atómico

O peso relativo de um átomo de um dado elemento é expressa com base num padrão (constante de peso atómico - 1/12 do peso de um átomo de carbono, mu = 1,66057*10-27 kg). Desta forma, permite a atribuição dos pesos atómicos em unidades de 1 a aproximadamente 300. Com aumento do número protões aumenta também geralmente o peso atómico.

Electronegatividade

Consiste na tendência que um átomo apresenta para atrair os electrões da ligação. Este valor é característico para cada elemento químico e permite aos químicos a avaliação do comportamento dos elementos químicos em determinados tipos de reacções e também das propriedades dos seus compostos.

Descrição

Informação geral sobre um elemento químico, por exemplo a sua cor, reactividade, classificação como metais, não-metais, semi-metais e gases.

Inspecção da estrutura

Formato ChemDraw binary

Formato binário usado pelo popular programa ChemDraw e usado também por uma série de outros editores moleculares.

Formato ChemDraw CDXML

Versão XML do formato binário acima descrito.

Formato padrão XML para a descrição das moléculas que é frequentemente usado na Internet.

InChI (Identificador químico internacional)

Formato linear apropriado para a atribuição inequívoca do “nome" das moléculas numa forma legível em computador. Recentemente, tem sido cada vez mais usado na Internet.

Molfile

Provavelmente o formato padrão mais amplamente utilizado para escrever a estrutura molecular. É usado por muitas bases de dados como o formato de conversão e está frequentemente disponível na Internet.

SMILES (Simplified Molecular Input Line Entry System)

Formato linear semelhante à notação clássica para a descrição de moléculas químicas. Foi desenvolvido para ser simultaneamente compreensível e editável por utilizadores, por exemplo CC=O (etanal). Este formato torna possível criar uma descrição de uma estrutura molecular sem a necessidade de um editor molecular e consequentemente é apropriado para o uso em texto e na Internet.

Fórmula molecular

Descreve o tipo e o número de átomos numa molécula de um dado composto. A sua escrita é levada a cabo como uma série de símbolos dos elementos químicos e a sua respectiva contagem. Não permite a descrição da estrutura interna da molécula, apenas da sua composição.

Formato de banco de dados de proteína

Formato de texto para a descrição de macromoléculas biológicas (proteínas, ácidos nucleicos).

Carga

É a diferença entre o número de electrões e protões numa molécula – quando existem mais electrões do que protões, a carga é negativa, de outra forma é positiva.

Peso molecular

É a soma dos pesos atómicos de todos os átomos na molécula.

Ligação simples

É a ligação formada por um par de electrões, representa-se por uma simples linha (-). A ordem da ligação é um factor muito importante na determinação da reactividade das ligações e noutras propriedades.

Ligação dupla

É formada por dois pares de electrões, representa-se por duas linhas paralelas (=).

Ligação Tripla

É formada por três pares de electrões, representa-se por três linhas paralelas (≡).

Grupo Funcional

É um átomo ou um grupo de átomos com estrutura específica que origina propriedades características às moléculas, tais como a reactividade em alguns tipos de reacções. A identificação dos grupos funcionais é imprescindível para a compreensão das propriedades químicas de uma molécula.
